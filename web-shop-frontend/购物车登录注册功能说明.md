### 3. 注册页面（Register.vue）

#### 页面特点
- 🎨 **渐变背景** - 粉红渐变效果
- 📝 **完整验证** - 多重输入验证
- 🔒 **密码确认** - 两次密码比对
- 📱 **手机号验证** - 格式校验
- ✅ **协议确认** - 用户协议勾选

#### 核心功能

**1. 表单验证**
```javascript
const registerRules = {
  username: [
    { required: true, message: '请输入用户名' },
    { min: 3, max: 20, message: '用户名长度在 3 到 20 个字符' },
    { pattern: /^[a-zA-Z0-9_]+$/, message: '只能包含字母、数字和下划线' }
  ],
  phone: [
    { required: true, validator: validatePhone }
  ],
  password: [
    { required: true, message: '请输入密码' },
    { min: 6, max: 20, message: '密码长度在 6 到 20 个字符' }
  ],
  confirmPassword: [
    { required: true, validator: validateConfirmPassword }
  ],
  agreement: [
    { required: true, validator: validateAgreement }
  ]
}
```

**2. 手机号验证**
```javascript
const validatePhone = (rule, value, callback) => {
  if (!/^1[3-9]\d{9}$/.test(value)) {
    callback(new Error('请输入正确的手机号'))
  } else {
    callback()
  }
}
```

**3. 密码确认验证**
```javascript
const validateConfirmPassword = (rule, value, callback) => {
  if (value !== registerForm.password) {
    callback(new Error('两次输入的密码不一致'))
  } else {
    callback()
  }
}
```

**4. 注册逻辑**
```javascript
const handleRegister = async () => {
  // 1. 验证表单
  await registerFormRef.value.validate()
  
  // 2. 调用注册API
  await userStore.register({

    password: registerForm.password
  })
  
  // 3. 注册成功后跳转登录页
  setTimeout(() => {
    goToLogin()
  }, 1500)
}
```

## 📊 API接口说明

### 购物车API（src/api/cart.js）

**启用/关闭Mock数据：**
```javascript
const MOCK_ENABLED = true // true=使用Mock，false=使用真实API
```

**接口列表：**

1. **获取购物车列表**
   ```javascript
   GET /cart
   getCartList()
   ```

2. **添加到购物车**
   ```javascript
   POST /cart
   addToCart(data)
   
   // 参数
   {
     productId: 商品ID,
     name: 商品名称,
     price: 单价,
     image: 图片,
     specs: 规格信息,
     quantity: 数量,
     stock: 库存
   }
   ```

3. **更新商品数量**
   ```javascript
   PUT /cart/:id
   updateCartItem(id, { quantity })
   ```

4. **删除商品**
   ```javascript
   DELETE /cart/:id
   deleteCartItem(id)
   ```

5. **批量删除**
   ```javascript
   POST /cart/batch-delete
   batchDeleteCartItems(ids)
   ```

6. **切换选中状态**
   ```javascript
   PUT /cart/:id/toggle
   toggleCartItem(id, checked)
   ```

7. **全选/取消全选**
   ```javascript
   PUT /cart/toggle-all
   toggleAllCartItems(checked)
   ```

8. **清空购物车**
   ```javascript
   POST /cart/clear
   clearCart()
   ```

9. **获取购物车数量**
   ```javascript
   GET /cart/count
   getCartCount()
   ```

## ⚠️ 注意事项

### Mock数据限制

1. **购物车数据**
   - 存储在localStorage
   - 刷新页面不会丢失
   - 清除浏览器数据会丢失

2. **用户数据**
   - 仅支持预设的测试账号
   - 注册不会真正保存用户
   - Token仅为模拟标识

3. **数据同步**
   - Mock数据仅在本地存储
   - 切换浏览器数据不同步
   - 多标签页数据可能不同步

### 开发建议

1. **localStorage管理**
   ```javascript
   // 清除所有Mock数据
   localStorage.clear()
   
   // 查看购物车数据
   localStorage.getItem('mock_cart_data')
   
   // 查看用户信息
   localStorage.getItem('user_info')
   localStorage.getItem('user_token')
   ```

2. **调试技巧**
   - 打开浏览器控制台查看日志
   - 所有操作都有详细的console输出
   - 使用Vue DevTools查看状态

3. **测试账号**
   - 使用提供的测试账号登录
   - 不要尝试注册后立即登录（Mock不保存）
