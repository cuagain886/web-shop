# 用户个人中心功能说明

## 📋 功能概览

用户个人中心是用户管理个人信息、订单、收藏、评价等的核心功能区域。根据用户角色（普通用户/商家）自动跳转到对应的个人中心页面。

---

## 🎯 路由逻辑

### Header 组件智能路由

点击顶部导航栏的"个人中心"时，根据用户角色自动跳转：

```javascript
// src/components/layout/Header.vue
case 'profile':
  if (userStore.userInfo?.role === 'merchant') {
    router.push('/admin/profile')  // 商家 → 管理端个人信息页
  } else {
    router.push('/profile')         // 普通用户 → 用户端个人中心
  }
  break
```

**路由地址**：
- 普通用户：`/profile` → `UserProfile.vue`
- 商家用户：`/admin/profile` → `AdminProfile.vue`

## 一、用户信息展示区

### 1.1 个人资料卡片

**位置**：左侧导航栏顶部

**展示内容**：
- **头像**：80x80 圆形头像（支持自定义或显示用户名首字母）
- **用户名**：显示昵称或用户名
- **会员等级**：根据积分自动计算
  - 钻石会员：≥5000分
  - 黄金会员：≥2000分
  - 白银会员：≥500分
  - 普通会员：<500分
- **身份标签**：
  - 已绑定手机（绿色 ✓）
  - 未绑定手机（灰色提示）

**代码实现**：
```vue
const levelInfo = computed(() => {
  const points = assets.points
  if (points >= 5000) return { name: '钻石会员', type: '' }
  else if (points >= 2000) return { name: '黄金会员', type: 'warning' }
  else if (points >= 500) return { name: '白银会员', type: 'info' }
  else return { name: '普通会员', type: 'info' }
})
```

## 七、安全设置（SecuritySettings.vue）

### 7.1 修改密码

**表单字段**：

| 字段 | 验证规则 | 说明 |
|------|----------|------|
| 当前密码 | 必填 | 验证旧密码 |
| 新密码 | 必填，6-20位，不能与旧密码相同 | 密码强度要求 |
| 确认新密码 | 必填，与新密码一致 | 防止输入错误 |

**自定义验证器**：
```javascript
// 验证新密码
const validateNewPassword = (rule, value, callback) => {
  if (value === '') {
    callback(new Error('请输入新密码'))
  } else if (value.length < 6 || value.length > 20) {
    callback(new Error('密码长度为6-20位'))
  } else if (value === passwordForm.oldPassword) {
    callback(new Error('新密码不能与旧密码相同'))
  } else {
    callback()
  }
}
```

## 🔗 路由配置

### userRoutes.js 新增路由

```javascript
{
  path: '/profile',
  name: 'UserProfile',
  component: () => import('@/pages/user/UserProfile.vue'),
  meta: { title: '个人中心', requiresAuth: true }
}
```

**路由守卫**：
- `requiresAuth: true` - 需要登录才能访问
- 未登录用户访问会自动跳转到 `/login?redirect=/profile`
- 登录成功后自动返回个人中心
