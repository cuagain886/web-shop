## ✅ 已实现功能

### 1. 页面结构

#### 面包屑导航
- 首页 / 商品详情 / 商品名称

**右侧 - 商品信息：**
- 商品标题和副标题

- 规格选择（颜色、容量等）
- 库存显示和数量选择
- 操作按钮（立即购买、加入购物车）
- 服务承诺（正品保证、极速物流、7天退换）

#### 商品详情Tab页
- **商品详情**：图文介绍
- **规格参数**：参数表格
- **售后保障**：售后服务说明

```javascript
  // 检查库存

```

**规格特点：**
- 支持多维度规格（颜色、容量、内存等）
- 无库存规格自动禁用并标记
- 选择规格后动态更新价格
- 必须选择所有规格才能购买

#### 库存管理
```javascript
// 动态计算当前库存
const currentStock = computed(() => {
  if (selectedSpec.value.stock !== undefined) {
    return selectedSpec.value.stock
  }
  return product.value.stock || 0
})
```

**库存提示：**
- 库存充足：绿色显示
- 库存紧张（<10）：橙色显示 + 提示
- 无库存：灰色显示 + 按钮禁用

#### 数量选择
- 使用Element Plus的InputNumber组件
- 最小值：1
- 最大值：当前库存
- 无库存时禁用
