# 商品列表页功能说明

## 📋 功能概览

商品列表页（ProductList）是用户浏览和筛选商品的核心页面，支持多维度筛选、搜索、排序、视图切换和分页等功能。

---

## 🎯 页面入口

### 1. 主页分类导航入口

**位置**：主页左侧分类导航栏

**交互**：点击任意分类（如"手机数码"、"电脑办公"）

**跳转逻辑**：
```javascript
// Home.vue
const goToCategory = (categoryId) => {
  router.push({
    path: '/products',
    query: { categoryId }
  })
}
```

**示例**：
- 点击"手机数码" → 跳转到 `/products?categoryId=2`
- 自动筛选显示该分类下的所有商品

### 2. 直接访问

**URL**：`http://localhost:5174/products`

**支持的查询参数**：
- `categoryId`：分类ID，自动应用分类筛选
- `keyword`：搜索关键词，自动填充搜索框

## 🔍 筛选与搜索功能

### 1. 分类筛选

**UI组件**：`el-radio-group` + `el-radio-button`

**选项**：
- 全部分类（默认）
- 10个分类按钮

**筛选逻辑**：
```javascript
if (filters.categoryId) {
  result = result.filter(p => p.categoryId === filters.categoryId)
}
```

**交互特性**：
- 选择分类后立即筛选
- 自动重置到第1页

### 2. 价格区间筛选

**预设区间**（单选）：
- 不限（默认）
- 0-100元
- 100-500元
- 500-1000元
- 1000-5000元
- 5000元以上

**筛选逻辑**：
```javascript
if (filters.priceRange === '0-100') return price >= 0 && price <= 100
if (filters.priceRange === '100-500') return price > 100 && price <= 500
if (filters.priceRange === '500-1000') return price > 500 && price <= 1000
if (filters.priceRange === '1000-5000') return price > 1000 && price <= 5000
if (filters.priceRange === '5000+') return price > 5000
```

### 3. 自定义价格区间

**UI组件**：两个 `el-input` + "确定"按钮

**输入字段**：
- 最低价（`filters.minPrice`）
- 最高价（`filters.maxPrice`）

**验证规则**：
- 最低价不能大于最高价
- 输入自定义价格时，自动清空预设区间

**筛选逻辑**：
```javascript
if (filters.minPrice !== null && filters.minPrice !== '') {
  result = result.filter(p => p.price >= filters.minPrice)
}
if (filters.maxPrice !== null && filters.maxPrice !== '') {
  result = result.filter(p => p.price <= filters.maxPrice)
}
```

### 4. 关键词搜索

**UI组件**：`el-input` 带搜索按钮

**触发方式**：
- 点击"搜索"按钮
- 按下回车键（`@keyup.enter`）

**匹配规则**：模糊匹配商品名称或描述
```javascript
if (filters.keyword) {
  const keyword = filters.keyword.toLowerCase()
  result = result.filter(p => {
    const name = (p.name || '').toLowerCase()
    const desc = (p.desc || p.description || '').toLowerCase()
    return name.includes(keyword) || desc.includes(keyword)
  })
}
```

**示例**：
- 搜索 "iPhone" → 匹配 "iPhone 15 Pro Max"、"iPhone 15"、"iPhone SE"等
- 搜索 "芯片" → 匹配描述中含"芯片"的商品

### 5. 组合筛选

**筛选顺序**：分类 → 价格 → 关键词 → 排序

**示例**：
1. 选择"手机数码"分类
2. 选择"1000-5000元"价格区间
3. 搜索"iPhone"
4. 结果：1000-5000元价格区间内的iPhone系列手机

## 🎨 排序功能

**排序选项**（下拉选择）：
1. **综合排序**（默认）：保持原始顺序
2. **价格从低到高**：`price` 升序
3. **价格从高到低**：`price` 降序
4. **销量从高到低**：`sales` 降序

**排序逻辑**：
```javascript
if (sortType.value === 'price-asc') {
  result.sort((a, b) => a.price - b.price)
} else if (sortType.value === 'price-desc') {
  result.sort((a, b) => b.price - a.price)
} else if (sortType.value === 'sales-desc') {
  result.sort((a, b) => (b.sales || 0) - (a.sales || 0))
}
```

## 📄 分页功能

### 分页配置

**默认配置**：
- 当前页：第 1 页
- 每页显示：10 条
- 可选每页条数：10、20、30、50

### 分页组件

**Element Plus Pagination**：
```vue
<el-pagination
  v-model:current-page="pagination.page"
  v-model:page-size="pagination.pageSize"
  :page-sizes="[10, 20, 30, 50]"
  :total="total"
  layout="total, sizes, prev, pager, next, jumper"
  @size-change="handleSizeChange"
  @current-change="handlePageChange"
/>
```

**布局说明**：
- `total`：显示总条数（"共找到 XX 件商品"）
- `sizes`：每页条数选择器
- `prev/pager/next`：上一页/页码/下一页
- `jumper`：跳转到指定页

### 分页逻辑

**计算当前页商品**：
```javascript
const displayProducts = computed(() => {
  total.value = filteredProducts.value.length
  const start = (pagination.page - 1) * pagination.pageSize
  const end = start + pagination.pageSize
  return filteredProducts.value.slice(start, end)
})
```

**页面变化时**：
- 自动滚动到页面顶部（平滑滚动）
- 重新渲染商品列表

**每页条数变化时**：
- 重置到第 1 页
- 重新计算总页数

## 🚀 核心功能实现

### 1. 路由参数初始化

**场景**：从主页点击分类跳转

**实现**：
```javascript
const initFromRoute = () => {
  const categoryId = route.query.categoryId
  if (categoryId) {
    filters.categoryId = parseInt(categoryId)
  }

  const keyword = route.query.keyword
  if (keyword) {
    filters.keyword = keyword
  }
}
```

**监听路由变化**：
```javascript
watch(() => route.query, (newQuery) => {
  if (route.path === '/products') {
    initFromRoute()
  }
}, { deep: true })
```

### 2. 计算属性链式过滤

**优势**：
- 自动响应筛选条件变化
- 不需要手动触发更新
- 支持多条件组合

**实现**：
```javascript
const filteredProducts = computed(() => {
  let result = [...allProducts.value]
  
  // 1. 分类筛选
  if (filters.categoryId) { /* ... */ }
  
  // 2. 价格筛选
  if (filters.priceRange) { /* ... */ }
  if (filters.minPrice) { /* ... */ }
  if (filters.maxPrice) { /* ... */ }
  
  // 3. 关键词搜索
  if (filters.keyword) { /* ... */ }
  
  // 4. 排序
  if (sortType.value) { /* ... */ }
  
  return result
})
```

### 3. 分页计算

**显示商品 = 过滤后的商品切片**：
```javascript
const displayProducts = computed(() => {
  total.value = filteredProducts.value.length
  const start = (pagination.page - 1) * pagination.pageSize
  const end = start + pagination.pageSize
  return filteredProducts.value.slice(start, end)
})
```
