# 项目说明

## 项目概述

这是一个采用京东风格设计的电商前端展示项目，使用 Vue 3 + Element Plus 构建。

## 当前版本特点

### ✅ 已实现

1. **首页展示** (`src/pages/product/Home.vue`)
   - 左侧分类导航（10个主要商品分类）
   - 中间轮播图（4个促销广告，使用渐变色彩）
   - 右侧信息栏（用户登录、公告信息）
   - 限时秒杀区域（5个秒杀商品）
   - 推荐商品区域（8个推荐商品，4列布局）

2. **页面布局**
   - 顶部导航栏 (`src/components/layout/Header.vue`)
   - 底部信息栏 (`src/components/layout/Footer.vue`)
   - 响应式设计，适配移动端

3. **路由系统**
   - 基础路由配置 (`src/router/index.js`)
   - 用户端路由 (`src/router/userRoutes.js`)
   - 404 页面重定向

### 📦 保留的架构文件

以下文件当前版本暂未使用，但作为完整架构保留，便于后续扩展：

- `src/api/` - API 请求封装（user.js, product.js, order.js, cart.js）
- `src/stores/` - Pinia 状态管理（userStore, cartStore, productStore）
- `src/hooks/` - 自定义组合式函数（useAuth, useCart）
- `src/utils/` - 工具函数（request, auth, format）

## 技术细节

### 组件开发规范

1. 使用 `<script setup>` 语法
2. 每个组件添加控制台日志，方便调试
3. 样式使用 `scoped` 避免污染
4. 使用 Element Plus 组件库

### 样式规范

1. 容器宽度：`1200px`
2. 圆角：`4px`
3. 阴影：`box-shadow: 0 2px 8px rgba(0,0,0,0.1)`
4. 过渡动画：`transition: 0.3s`

### 响应式断点

```css
@media (max-width: 1200px) {
  /* 移动端适配 */
}
```

## 开发调试

### 控制台日志说明

项目中添加了详细的控制台日志：

- `🚀` - 应用初始化
- `📦` - 模块注册
- `✅` - 操作成功
- `❌` - 错误信息
- `📍` - 路由跳转
- `📌` - 组件加载
- `📊` - 数据统计
- `🎉` - 页面加载

### 常见问题

1. **页面空白**
   - 检查浏览器控制台是否有错误
   - 确认 Element Plus 已正确安装
   - 清除浏览器缓存后重新加载

2. **样式不显示**
   - 确认 `element-plus/dist/index.css` 已引入
   - 检查 scoped 样式是否正确

3. **路由不工作**
   - 检查 `router/index.js` 配置
   - 确认组件路径正确

## 部署说明

### 开发环境

```bash
npm run dev
```

### 生产构建

```bash
npm run build
```

构建产物在 `dist/` 目录下。

### 部署到服务器

1. 上传 `dist/` 目录到服务器
2. 配置 Nginx 或 Apache
3. 设置正确的路由重定向（用于 SPA）
