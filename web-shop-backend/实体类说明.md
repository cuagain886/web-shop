## 🔧 使用的注解

### Lombok 注解
- `@Data` - 自动生成 getter/setter/toString/equals/hashCode

### MyBatis-Plus 注解
- `@TableName` - 指定对应的数据库表名
- `@TableId` - 标识主键，使用自增策略
- `@TableLogic` - 标识逻辑删除字段
- `@TableField(fill = FieldFill.INSERT)` - 插入时自动填充
- `@TableField(fill = FieldFill.INSERT_UPDATE)` - 插入和更新时自动填充

## 📝 字段映射规则

### 1. 驼峰命名转换
MyBatis-Plus 自动将驼峰命名转换为下划线命名：
```java
// Java 字段
private String userName;

// 对应数据库字段
user_name
```

已在 `application.properties` 配置：
```properties
mybatis-plus.configuration.map-underscore-to-camel-case=true
```

### 2. 时间类型映射
```java
// Java 8 时间类型
LocalDateTime  ->  MySQL datetime
LocalDate      ->  MySQL date
```

### 3. 数值类型映射
```java
// 金额字段
BigDecimal     ->  MySQL decimal(10,2)

// 整数字段
Integer        ->  MySQL int
Long           ->  MySQL bigint

// 布尔字段（用 Integer）
Integer        ->  MySQL tinyint (0/1)
```

## ⚙️ 字段自动填充

### 配置类：`MyBatisMetaObjectHandler`

**自动填充字段**：
- `createdTime` - 插入时填充当前时间
- `updatedTime` - 插入和更新时填充当前时间
- `viewTime` - 插入时填充当前时间（浏览历史）

**使用示例**：
```java
// 插入数据时，createdTime 和 updatedTime 自动填充
User user = new User();
user.setUsername("test");
user.setPassword("123456");
userMapper.insert(user);
// createdTime 和 updatedTime 自动设置为当前时间

// 更新数据时，updatedTime 自动更新
user.setNickname("新昵称");
userMapper.updateById(user);
// updatedTime 自动更新为当前时间
```

## 🗑️ 逻辑删除

### 配置
已在 `application.properties` 配置：
```properties
mybatis-plus.global-config.db-config.logic-delete-field=deleted
mybatis-plus.global-config.db-config.logic-delete-value=1
mybatis-plus.global-config.db-config.logic-not-delete-value=0
```

### 支持逻辑删除的实体类
带 `@TableLogic` 注解的字段：
- User
- UserAddress
- ProductCategory
- Product
- Orders
- ProductReview
- Refund

### 使用效果
```java
// 删除用户（逻辑删除）
userMapper.deleteById(1L);
// 实际执行：UPDATE user SET deleted=1 WHERE id=1 AND deleted=0

// 查询用户（自动过滤已删除）
User user = userMapper.selectById(1L);
// 实际执行：SELECT * FROM user WHERE id=1 AND deleted=0

// 查询列表（自动过滤已删除）
List<User> users = userMapper.selectList(null);
// 实际执行：SELECT * FROM user WHERE deleted=0
```

## 📦 使用示例

### 1. 基础 CRUD

```java
// 插入
User user = new User();
user.setUsername("test");
user.setPassword("encrypted_password");
user.setRole("user");
userMapper.insert(user);

// 根据 ID 查询
User user = userMapper.selectById(1L);

// 条件查询
QueryWrapper<User> wrapper = new QueryWrapper<>();
wrapper.eq("username", "test");
User user = userMapper.selectOne(wrapper);

// 更新
user.setNickname("新昵称");
userMapper.updateById(user);

// 删除（逻辑删除）
userMapper.deleteById(1L);
```

### 2. 复杂查询

```java
// 分页查询商品
Page<Product> page = new Page<>(1, 10);
QueryWrapper<Product> wrapper = new QueryWrapper<>();
wrapper.eq("status", 1)  // 上架状态
       .eq("is_hot", 1)  // 热销商品
       .orderByDesc("sales");  // 按销量排序
Page<Product> result = productMapper.selectPage(page, wrapper);

// 关联查询（需要自定义 SQL）
// 在 Mapper 接口中定义方法，在 XML 中编写 SQL
List<OrderDetailVO> orders = orderMapper.selectOrdersWithItems(userId);
```

### 3. Lambda 查询（推荐）

```java
// Lambda 查询，避免字段名写错
LambdaQueryWrapper<Product> wrapper = new LambdaQueryWrapper<>();
wrapper.eq(Product::getStatus, 1)
       .eq(Product::getCategoryId, categoryId)
       .between(Product::getPrice, minPrice, maxPrice)
       .orderByDesc(Product::getSales);
List<Product> products = productMapper.selectList(wrapper);
```

## ⚠️ 注意事项

### 1. 订单实体类名
```java
// ❌ 错误
public class Order { }

// ✅ 正确
public class Orders { }
```
**原因**: 避免与 SQL 关键字冲突，类名统一为 `Orders`

### 2. 时间字段不要手动赋值
```java
// ❌ 错误
user.setCreatedTime(LocalDateTime.now());
user.setUpdatedTime(LocalDateTime.now());

// ✅ 正确（自动填充）
// 什么都不做，MyBatis-Plus 自动填充
```

### 3. 逻辑删除字段不要手动操作
```java
// ❌ 错误
user.setDeleted(1);
userMapper.updateById(user);

// ✅ 正确
userMapper.deleteById(userId);
```

### 4. BigDecimal 金额字段
```java
// ❌ 错误（精度丢失）
product.setPrice(new BigDecimal(99.99));

// ✅ 正确
product.setPrice(new BigDecimal("99.99"));
```

### 5. JSON 字段序列化
```java
// 使用 Jackson 序列化
ObjectMapper mapper = new ObjectMapper();

// 商品图片
List<String> images = Arrays.asList("img1.jpg", "img2.jpg");
String imagesJson = mapper.writeValueAsString(images);
product.setImages(imagesJson);

// 反序列化
List<String> images = mapper.readValue(product.getImages(), 
    new TypeReference<List<String>>(){});
```
